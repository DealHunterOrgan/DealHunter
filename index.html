¿Por qué es obligatorio el index.html?

Como indica el propio error: "For root URLs you must provide an index.html file".

Es el estándar de internet: cuando visitas la raíz de un sitio web, el servidor busca automáticamente un archivo llamado index.html para mostrar el contenido inicial.

Sin ese archivo, el navegador intentaría listar los archivos de la carpeta, pero GitHub bloquea esa vista por seguridad, lo que termina mostrando un error 404.
